# react-reactive-forms

## Commit messages
- No Claude/Anthropic credits in commit messages.
- Never mention CLAUDE.md updates in commit messages.

## Branching
- Never push to main or master.
- Every fix, feature, or chore must live on its own dedicated branch.
- Branch naming: `devname/(fix|feat|chore)-slug` (e.g. `aobayos/fix-sideeffects`).
- Never stack commits across multiple branches at the same time — all changes go to the current branch.
- If a commit has already been pushed but the branch is not yet merged, amend it instead of adding a new commit.
